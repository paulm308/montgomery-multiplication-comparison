# montgomery-multiplication-comparison
## Summary
This repository contains Python implementations of several Montgomery modular 
multiplication algorithms, based on the paper "Analyzing and Comparing Montgomery 
Multiplication Algorithms" by Koç, Acar, and Kaliski (IEEE Micro, 1996). The project 
was created as part of a seminar on computer networks, in which the 
underlying paper was presented.

Implemented algorithms:
-  Separated Operand Scanning (SOS)
- Coarsely Integrated Operand Scanning (CIOS)
- Finely Integrated Operand Scanning (FIOS)
- Finely Integrated Product Scanning (FIPS)
- Coarsely Integrated Hybrid Scanning (CIHS)

## Reference
Koç, Ç. K., Acar, T., & Kaliski, B. S. (1996). Analyzing and comparing Montgomery 
multiplication algorithms. IEEE Micro, 16(3), 26-33.
